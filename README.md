# Android App with Real‑Time AI Voice Interaction

A lightweight Android application enabling **real‑time voice interaction with an AI assistant**.  
The app listens to user speech, sends it to an AI model running in Live mode, receives actionable instructions, and **automatically performs UI actions** based on the AI’s guidance.

## Features
- **Real‑time AI voice interaction**  
  Continuous voice input with instant AI responses.
- **Automatic UI actions**  
  The AI tells the app what to press — the app executes it.
- **WebRTC calling**  
  Built‑in audio/video call support.
- **Socket.IO real‑time messaging**  
  Fast, event‑driven chat communication.
- **Firebase Authentication**  
  Secure user login and identity management.

## Tech Stack
- Kotlin / Android
- WebRTC
- Socket.IO
- Firebase Auth
- AI Live Voice Processing

## Purpose
Exploring hands‑free Android interaction using AI‑driven voice commands for accessibility, automation, and experimental AI‑assisted UX.
